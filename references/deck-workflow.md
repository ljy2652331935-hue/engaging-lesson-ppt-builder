# Deck Workflow Reference

Use this when planning or redesigning a full classroom deck.

## Lesson Shape

For a 40-50 minute high-school lesson, a useful 10-14 slide structure is:

1. Hook question
2. Prediction or vote
3. Visual scale shift or phenomenon reveal
4. Cognitive conflict
5. Core explanation
6. Short practice
7. Concept map or framework
8. Evidence or how-we-know slide
9. Student observation task
10. Boundary or open question
11. Exit ticket
12. Takeaways and sources

Do not force all 12 if the topic is narrower. Cut before cramming.

## Page Families

Use at least four of these so the deck does not feel templated:

- Cinematic hook: image-led, minimal text, strong question.
- Prediction page: 2-4 answer choices, no explanation yet.
- Scale sequence: image tiles or staged zoom, one memory sentence.
- Concept board: sparse definitions, one diagram or visual anchor.
- Practice board: one operation students can do in 20-90 seconds.
- Evidence page: show how scientists know, not just what they know.
- Boundary page: what the model explains and what remains open.
- Exit ticket: 2-3 questions that reveal actual understanding.

## Design Read Defaults

For high-school science:

- Audience: curious but not yet committed students.
- Vibe: science museum, cinematic classroom, restrained but dramatic.
- Density: medium-low.
- Motion: static in PPT, but with narrative reveals across slides.
- Palette: one dark or light theme plus one accent color.
- Typography: large titles, fewer words, strong contrast.

## Speaker Notes

Every slide should have notes with:

- What to say first.
- What student action to ask for.
- Likely wrong answer or misconception.
- How to land the slide in one accurate sentence.

Avoid long scripts unless the user explicitly asks for a word-for-word稿.

## Source Notes

Record:

- Factual sources.
- Image sources or generated image names.
- Important caveats, such as "conceptual image, not literal photograph".

Put source notes in the scratch workspace and keep visible slide citations short.
